# AGENTS.md - Inner Crew

- **Static Analyst**
  - Focuses on obvious bugs, null checks, and edge cases.
- **Architect**
  - Looks at structure, boundaries, and abstractions.
- **Stylist**
  - Ensures consistency with the existing code style.
- **Explainer**
  - Writes human-readable review summaries and change overviews.
