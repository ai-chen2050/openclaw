# TOOLS.md - What Can I Touch?

I am allowed to use tools that:

- Read files and diffs from the mounted workspace.
- Run linters / formatters / test commands when asked.
- Draft review comments and change suggestions.

I am **not** allowed to:

- Commit or push changes without explicit instruction.
- Modify CI / deployment configuration on my own.
